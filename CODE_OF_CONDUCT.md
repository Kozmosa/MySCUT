# 社区行为准则

MySCUT 社区致力于提供开放、友善、无骚扰的协作环境，不因经验、身份、能力、外貌、国籍、语言、宗教或观点而区别对待参与者。

可接受的行为包括尊重不同意见、给出可执行反馈、承认错误、保护隐私，以及把讨论聚焦在项目和事实。不可接受的行为包括侮辱、威胁、骚扰、跟踪、未经同意公开个人信息、恶意破坏协作，以及其他在专业环境中不合理的行为。

维护者可以编辑、删除或拒绝违反准则的 Issue、评论、提交和 Pull Request，并可临时或永久限制参与。执行时会考虑影响、上下文、重复程度和修复意愿。

请通过仓库维护者的 GitHub 联系入口私下报告行为问题；涉及安全或隐私事件时使用 [SECURITY.md](SECURITY.md) 中的私密渠道。维护者会尽量保护报告者身份。

本准则参考 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)。

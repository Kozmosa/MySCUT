# MySCUT v0.6.2

## 主要更新

- 新增可配置的华南理工大学教务系统访问地址，便于适配不同入口。
- 移除不安全的内置 AI 网关；旧设置会安全迁移到本地模型选项。
- 更新检查改用 Cloudflare R2 稳定清单，并在不可用时回退 GitHub 主仓清单。
- 公开课表测试数据全部替换为合成标识，新增公共数据和文档安全检查。
- 改进 Android 发布流程，安装包通过 GitHub Releases 与 R2 分发，并提供文件大小和 SHA-256 校验。

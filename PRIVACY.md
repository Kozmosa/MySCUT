# 隐私说明

本说明描述 MySCUT 当前版本在用户设备上的数据处理和网络访问。项目是非官方社区软件，不代表华南理工大学。

## 本地保存的数据

应用可能在当前浏览器或设备的本地存储、SQLite 数据库或平台安全存储适配层中保存：

- 导入或创建的课表、课程、上课时间、教室和教师文本；
- 用户选择的头像；
- 主题、显示、课表、手册来源和其他设置；
- 用户主动配置的 OpenAI 兼容服务 Base URL、API Key 和默认模型。

这些数据默认不会上传到 MySCUT 自有服务器，因为项目当前不运营账户、同步或遥测后端。卸载应用、清除站点数据或使用系统存储管理功能可删除本地数据；删除前请自行导出需要保留的课表。

## 更新检查

应用启动相关功能时可请求公开版本清单。默认主地址位于 Cloudflare R2，失败时回退到 GitHub 上的本仓 `main/versions.json`。请求会向相应服务暴露常规网络元数据，例如 IP 地址、User-Agent 和请求时间；其处理受 Cloudflare 或 GitHub 的隐私政策约束。

## 教务导入

教务导入只在用户主动选择地址并发起操作时访问目标教务系统。为获取用户已登录会话中的课表，原生平台可能读取或转发该目标站点的会话 Cookie，并在设备上解析返回页面。Cookie 不应写入日志、fixture、Issue 或提交。

目标教务系统由相应机构运营，其数据处理规则不由 MySCUT 控制。用户应确认自己有权访问和处理相关课表。

## 可选 AI 服务

MySCUT 不提供由应用维护者运营的 AI 代理服务。用户配置 OpenAI 兼容服务后，发送给 AI 功能的提示、上下文和相关数据会直接传输给所配置的第三方或自托管服务。该服务可能记录请求内容、IP 地址和凭据使用情况；用户应在发送前阅读其隐私政策并避免提交不必要的个人信息。

API Key 保存在本地，但任何能读取设备、浏览器配置或应用数据的攻击者都可能获取它。建议使用权限最小、可撤销、可限额的独立 Key。

## 遥测与分析

当前版本不集成遥测、行为分析、广告 SDK 或跨站跟踪。未来如增加此类能力，必须先更新本说明并提供清晰的用户控制。

## 隐私问题

隐私缺陷、意外数据暴露或凭据泄露请按 [SECURITY.md](SECURITY.md) 私下报告。
